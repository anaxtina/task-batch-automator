# ⚙️ Task Batch Automator

A Python-based productivity utility designed to automate digital workspace setups and repetitive online routines through structured, category-based workflows.

## 🛠️ The Problem it Solves
Manual setup of daily digital environments (opening documentation, communication tools, and development environments) increases **Cognitive Load** and wastes valuable time. This tool reduces that friction to a single command.

## 🚀 Technical Highlights
- **Workflow Orchestration:** Implements a `RoutineAutomator` class for centralized management of digital assets.
- **Resource Management:** Dynamic mapping of URLs and local resources categorized by context (e.g., Study, Research, Admin).
- **Execution Stability:** Integrated `time-delay` logic to ensure system stability and prevent browser overhead during mass execution.
- **Type Safety:** Uses Python's `typing` module for robust and maintainable code.

## 💻 Tech Stack
- **Language:** Python 3.x
- **Core Modules:** `webbrowser`, `os`, `time`, `typing`

## 🧠 Behavioral Insight
This project is an application of **Habit Formation** and **Environmental Design** principles. By automating the "start" of a task, we lower the activation energy required for deep work, facilitating immediate focus.
